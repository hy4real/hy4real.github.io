神经元模型#

基本脑细胞被称为神经元，麦卡洛克和皮茨在论文中给出了高度简化的神经元模型。该模型（我们将其称为 MCP 神经元）在计算机科学领域具有重要意义。相互连接的 MCP 神经元群被称为人工神经网络。

MCP 神经元具有多个输入 $x_1, …, x_n$。这些输入代表了从神经元突触接收到的输入信号。因此，$x_i$ 可以是 $1$，对应于来自第 $i$ 个连接的输入信号的存在，或者 $x_i$ 是 $0$，对应于不存在来自第 $i$ 次连接的信号。MCP 神经元产生一个输出 $y$，它可以是 $1$，对应于发送信号的神经元，也可以是 $0$，对应于保持静止的神经元。数学上，MCP 神经元是一个函数，它接受 $1$ 和 $0$ 的 $n$ 元组，并产生 $1$ 或 $0$。为了量化每个突触对 MCP 神经元的影响，我们将权重 $w_i$ 分配给每个输入 $x_i$。权重是十进制数，权重的大小对应于相关连接对 MCP 神经元的影响程度。正权重对应于兴奋性突触，负权重对应于抑制性突触。每个输入都乘以相应的权重，然后将这些加权输入加在一起，得到

$$ w_1 \times x_1 + w_2 \times x_2 + … + w_n \times x_n $$

其中 $w_i \times x_i$ 表示 $w_i$ 乘以 $x_i$。这对应于假设由神经元执行的输入信号的求和。最后，有一个阈值 $T$，它是一个十进制数，与信号的加权和进行比较。这对应于接收信号之和必须超过的表观阈值，才能使神经元"放电"。如果

$$ w_1 \times x_1 + w_2 \times x_2 + … + w_n \times x_n \geq T $$

则 MCP 神经元产生 $1$ 的输出。如果

$$ w_1 \times x_1 + w_2 \times x_2 + … + w_n \times x_n < T $$

则 MCP 神经元产生 $0$。因此，MCP 神经元完全由其权重和阈值决定。通过选择不同的权重和阈值组合，我们可以产生几种不同的 MCP 神经元。

感知器#

感知器（Perceptron）是最简单的人工神经网络模型，由 Frank Rosenblatt 于 1957 年提出。

1. 什么是感知器？#

感知器是一种二元分类器，用于将输入数据分为两类。它模拟人脑神经元的工作方式，是神经网络的基本构建单元。

2. 感知器的工作原理#

感知器接收多个输入信号 $x_1, x_2, …, x_n$，每个输入都有一个对应的权重 $w_1, w_2, …, w_n$：

$$ \text{净输入} = \sum_{i=1}^{n} w_i x_i + b $$

其中 $b$ 是偏置（bias）项。

3. 激活函数#

感知器使用阶跃函数作为激活函数：

$$ y = \begin{cases} 1 & \text{如果 } \sum_{i=1}^{n} w_i x_i + b \geq 0 \ 0 & \text{如果 } \sum_{i=1}^{n} w_i x_i + b < 0 \end{cases} $$

4. 训练算法#

感知器通过以下规则学习：

  • 初始化：随机设置权重和偏置
  • 预测：计算输出并与真实标签比较
  • 更新：如果预测错误，调整权重

权重更新规则：

$$ w_i \leftarrow w_i + \alpha (y_{\text{真实}} - y_{\text{预测}}) x_i $$

其中 $\alpha$ 是学习率。

5. 局限性#

  • 只能解决线性可分问题
  • 无法学习 XOR 等非线性问题
  • 单层感知器能力有限

6. 多层感知器#

通过堆叠多个感知器层形成多层感知器（MLP），可以学习更复杂的非线性模式，这是深度神经网络的基础。
