保重身体

今天，一个初中同学联系我，我没想到。 我这个初中同学成绩不好，上课爱调皮捣蛋，着装也不像个学生，所以他经常被老师骂，而且平常的时候他喜欢借钱，借了很多同学的钱，也包括我。 作业批改后发下来订正，他第一时间来找我；有什么不会的题目，他也过来问我。 可能是因为他觉得我是个好学生，他在别的同学那里收获到的是嘲笑和排挤，只有我不这样，所以他一直对我不错。 他有的时候借我的钱，还拿这个钱给我买吃的，买文具送给我。毕业的前两个月，他没来上课，但是他在网上把他欠的钱都还给我了。 我不知道他有没有还别人的钱，因为他在别人那里风评不好，但是他确确实实把所有欠我的钱都还给我了，还多给了我五十块。 他本来在芜湖上初一。他的爸妈离婚了，他跟着他爸。 他爸有一个朋友就在合肥，家里孩子也上初中，我这个同学就重新念了初一，和叔叔的孩子在一个班，平常就寄住在这个叔叔家里，他经常挨他这个叔叔打。 他爸爸不常来看望他，来的时候除了给他一笔生活费，可能还有一顿打，有一次他来上课，我看到他的脖子后是青紫的印子。 他还是想念他妈妈，我后来听他在网上和我说，毕业前他没来上课的原因就是他回芜湖找他妈妈去了。 故事应该从他还完我的钱就结束了，但我没想到。 在我上了大学以后，他托人来加了我的微信。 第一句话就是找我借钱，说他现在在谈女朋友，需要花很多钱。 那个时候我相信他，我就借了他两三百，我说希望他有钱的时候还给我，后面有两次催他，他说他被网贷缠上了，每天都要拆东墙补西墙。 后面我气不过，可能也是那时候的我比较狭隘，我就把他的微信删了，这钱我不要了，我花二百块看清一个人。 从删掉他微信到昨天，他都没有试图重新联系我，今天突然给我发来一条好友申请，备注是他。 我本来想忽略的，但是我回想起之前的事情，是不是我做的不够厚道，在纠结中，我同意了他的好友申请。 我给他发了一个问号，他给我打来微信电话，说为什么我删除了他的微信，我用蹩脚的理由搪塞过去。 他问我现在在哪，我说在合肥，他说现在跟着他爸在做房地产，今天来合肥出差，问我下午有没有空，请他初中最好的朋友吃晚饭。 我以我不在市中心为理由，婉拒了他的邀请，如果我过去的话，我不知道怎么面对他：我还是学生，但是他已经闯社会好多年了，可以预见的是如同闰土和迅哥再次相见的尴尬。 我说：“你心意已经到了，不用你再请我吃晚饭了。” 事到如今，我也不想再去纠结是是非非了，无论是当时他没把钱还给我，还是我因此把他的微信删除，我觉得都不重要了。 初中那时的感情是真的，但是已经过去好多年了。其实，我觉得他自己心里也很清楚，我想远离他，大家都变了，话又从何说起呢，感情终究是变淡了。 我和他说起初中的事，都已恍如隔世。“那时候的你要面对的东西太多，把任何人放到你那时的处境，也不一定做得比你好。”他不说话。他问起我的现状，我把我在大学经历的起起伏伏笑着讲出来，他不相信，他说我成绩这么好，怎么会有这些烦恼苦闷。我说读书已经不是唯一的出路，大家都看得很开，他也认可。 我问他后面还有没有再回到初中，看以前的老师，他说回了四五次，当时只有班主任和政治老师对他不错，对他比较关心，他说请她们吃了很多次饭。 最后我跟他说注意身体，他说每天都要喝酒，脑子很晕，但是为了讨生活没办法，我再次跟他说保重身体，然后我们就告别。

深度学习入门 Day 4

神经元模型 基本脑细胞被称为神经元，麦卡洛克和皮茨在论文中给出了高度简化的神经元模型。该模型（我们将其称为 MCP 神经元）在计算机科学领域具有重要意义。相互连接的 MCP 神经元群被称为人工神经网络。 MCP 神经元具有多个输入 $x_1, …, x_n$。这些输入代表了从神经元突触接收到的输入信号。因此，$x_i$ 可以是 $1$，对应于来自第 $i$ 个连接的输入信号的存在，或者 $x_i$ 是 $0$，对应于不存在来自第 $i$ 次连接的信号。MCP 神经元产生一个输出 $y$，它可以是 $1$，对应于发送信号的神经元，也可以是 $0$，对应于保持静止的神经元。数学上，MCP 神经元是一个函数，它接受 $1$ 和 $0$ 的 $n$ 元组，并产生 $1$ 或 $0$。为了量化每个突触对 MCP 神经元的影响，我们将权重 $w_i$ 分配给每个输入 $x_i$。权重是十进制数，权重的大小对应于相关连接对 MCP 神经元的影响程度。正权重对应于兴奋性突触，负权重对应于抑制性突触。每个输入都乘以相应的权重，然后将这些加权输入加在一起，得到 $$ w_1 \times x_1 + w_2 \times x_2 + … + w_n \times x_n $$ 其中 $w_i \times x_i$ 表示 $w_i$ 乘以 $x_i$。这对应于假设由神经元执行的输入信号的求和。最后，有一个阈值 $T$，它是一个十进制数，与信号的加权和进行比较。这对应于接收信号之和必须超过的表观阈值，才能使神经元"放电"。如果 $$ w_1 \times x_1 + w_2 \times x_2 + … + w_n \times x_n \geq T $$ ...
